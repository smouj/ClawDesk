ClawDesk

Mission Control · Security-first · Loopback-only
v2.x

Dashboard local para operar OpenClaw con seguridad pro.

ClawDesk centraliza estado, logs y acciones del gateway en una interfaz moderna sin exponer tu máquina al exterior.

Instalar ahora Ver seguridad

Quickstart

One-liner remoto (stable)

curl -fsSL https://raw.githubusercontent.com/smouj/ClawDesk/main/scripts/install-remote.sh | bash

Instala la última release estable con verificación SHA256.

Git clone + install.sh

git clone https://github.com/smouj/ClawDesk.git
cd ClawDesk
bash install.sh

Ideal para contribuir o personalizar la instalación.

Características clave

Mission Control

Estado del gateway, latencia, token detectado y última actividad.

Acciones rápidas

Start/Stop/Restart con políticas de allow-actions.

Logs pro

Filtro, pausa, descarga y jerarquía visual clara.

Seguridad

ClawDesk escucha en loopback. El acceso remoto debe hacerse por túneles cifrados (Tailscale, WireGuard o SSH). Se bloquea 0.0.0.0 por defecto.

Loopback-only CORS allowlist Rate limit local

Comandos esenciales

clawdesk run
clawdesk status
clawdesk doctor
clawdesk open

WSL / Windows

En WSL, localhost se comparte con Windows. Abre el dashboard desde Windows en http://127.0.0.1:4178.
