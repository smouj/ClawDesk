ClawDesk

Instalación oficial (git clone + install.sh)

1) Clonar el repositorio

git clone https://github.com/smouj/ClawDesk.git
cd ClawDesk

2) Ejecutar el instalador

bash install.sh

El instalador detecta dependencias, configura OpenClaw y crea el comando clawdesk.

3) Comandos útiles

clawdesk run
clawdesk status
clawdesk doctor
clawdesk open
ClawDesk · instalación local · loopback only
